# WeatherWise-AI

WeatherWise AI is a full-stack weather intelligence platform designed to transform weather data into actionable insights. Users can search for weather information using cities, landmarks, postal codes, or coordinates and receive real-time weather conditions, forecasts, location information, and AI-powered recommendations.

## Features

* Real-time weather information
* 5-day weather forecast
* Current location weather detection
* AI-powered weather insights
* Travel Readiness Score
* Smart Packing Assistant
* Event Risk Detector
* Weather Impact Analysis
* Interactive maps and location explorer
* YouTube travel videos for searched locations
* Search history with full CRUD operations
* Data export in JSON, CSV, and Markdown formats
* Responsive design for desktop, tablet, and mobile devices

## Purpose

The goal of WeatherWise AI is not only to display weather data but to help users understand how weather conditions affect travel, outdoor activities, events, and daily life through intelligent recommendations and analytics.
